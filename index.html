我是index.html文件 1.git init 初始化本地仓库 2.git status 查看提交状态 3.git add index.html 将文件添加到暂存区 4.git status 5.git commit -m "提交信息" # 将本地仓库推送到远程仓库 6.在github上创建新的远程仓库 7.git push https://github.com/bruceyang6/git-demo.git master 8.为远程仓库起一个别名 origin git remote add origin https://github.com/bruceyang6/git-demo.git 重复第3、第5步后 9.git push origin master 重复第3、第5步后 10.git push -u origin master 记住远程仓库地址以及分支名称，下次提交，直接输入git push 即可。 11.
